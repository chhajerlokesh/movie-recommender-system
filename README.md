# 🎬 Movie Recommender System

A content-based movie recommendation system built with Python and Streamlit. This app suggests movies similar to the one selected by the user based on metadata such as genres, keywords, cast, and crew.

## 🚀 Features
* **Search Functionality:** Select a movie from a database of 5,000+ titles.
* **Vectorization:** Uses Bag of Words (CountVectorizer) and Cosine Similarity to find matches.
* **Interactive UI:** Built with Streamlit for a smooth user experience.
* **Poster Fetching:** (Optional) Integrates with TMDB API to show movie posters.
